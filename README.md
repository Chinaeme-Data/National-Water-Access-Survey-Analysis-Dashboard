# Project Title
National Water Access Survey Analysis Dashboard

## National-Water-Access-Survey-Analysis-Dashboard
This project analyzes water access data across rural and urban regions of Maji Ndogo to identify gaps in basic water availability. It uses Power BI to visualize trends and highlight areas that need infrastructure improvements and the cost of the improvements. The dashboard is designed for policymakers and NGOs.

## Preview
![Dashboard Screenshot](link-to-image)

## Key Features:
- Tracks basic vs below-basic water access
- Calculates population served per water source
- Highlights rural vs urban disparities
- Interactive filtering by province and source type

## Tech Stack
- Power BI
- DAX
- SQL (MySQL)
- Excel

## How to Use
1. Download the .pbix file from this repository
2. Open it using Power BI Desktop
3. Refresh the dataset if needed

## Data Source
The dataset includes:
- Water source type
- Infrastructure cost
- Project progress
- Number of people served
- Queue composition
- Location (town, province)
- Visits
Data was cleaned and transformed before analysis.

## Key Insights
- Rural areas have significantly lower access to basic water services
- Shared taps are the most common source in underserved regions
- Queue time strongly correlates with population served
